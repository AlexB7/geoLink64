geoLink 1.01 Release Notes (2010-04-26)

geoLink is a networked GEOS program written to support multiple application modules.  This version has only one application, an IRC client.

 

Known Issues in This Version

geoLink has problems connecting to some IRC servers. For NewNet (#c64friends), I've had the best luck with irc.tobug.net.

 

Requirements

a Commodore 64 running GEOS 2.0 (lower versions are not supported; higher versions will display a warning message)

a network card (tested with RR-NET, MMC Replay, FB-Net, 1541 Ultimate, and 64NIC+)

 

Usage

Double-click the geoLink icon.  The first time the program is run, it will not find a settings file, so it will display the setup dialog.  Enter your network information here, or click the DHCP checkbox.  In either case, you must provide a MAC address (it is suggested to use one that starts with the Commodore OUI of 00:80:10).  When setup is complete, select a name for the settings file, or click Cancel not to save.  You can save multiple settings file with different names; a selection dialog will appear when the program is started if more than one settings file is present.

 

To find out your network settings, you can select "options/setup" to display the setup dialog; just click the Cancel icon to dismiss it.

 

To test connectivity, click "options/ping" and enter an IP address or hostname to ping.  Results are shown in the dialog.

 

To start a chat, click "options/IRC".  Enter the IP address or hostname of the IRC server you want to connect to, along with the port (6667 by default) and a nickname.  geoLink will automatically send the USER and NICK commands; if login is successful, you will be able to join a channel and chat with other users by entering text in the input field at the bottom of the screen and hitting Enter or clicking the Send icon.

 

The following client commands are implemented:

/join           Join a channel (e.g. "join #c64friends"). You can only be in one channel at a time; if you are already in a channel when you use this command, geoLink will issue a /part command first.

/part           Leave the current channel.

/names     List nicks in the current channel.

/nick           Change your nickname (e.g. "/nick NewNick").

/me            Action command (e.g. "/me does something").

/msg          Private message (e.g. /msg Ponyboy private message"). Private responses are printed in italics; there is no facility for having a separate window open for private chat. In other words, you must prefix every message in a private conversation with "/msg".

 

In the text entry field, HOME will move to the beginning of the field and CLEAR will clear the field. You can also use F1 for home, and F7 for end.

 

An info dialog is available by clicking "geos/info", which shows the program version, the version of GEOS you are running, and the credits.
